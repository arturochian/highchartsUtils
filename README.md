highchartsUtils
===============

highchartsUtils is an R package, based on rCharts.
We focus here on the javascript libraries highcharts, highstocks and highmaps.

Installation
------------

```coffee
require(devtools)
install_github('highchartsUtils', 'bthieurmel')
```

Use
------------

```coffee
require(highchartsUtils)
?plotTimeSeries

```
